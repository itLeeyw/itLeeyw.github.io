2. Hexo配置

12. 进入「你的用户名.github.io」对应的 repo，打开 GitHub Pages 功能，如果已经打开了，你应该会看到一个预览链接

13. 访问这个连接，访问成功！

3.初始配置成功后再次创建博客

4. 注意
